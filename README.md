## 📖 这是什么？

Claude Code Skills 是 AI Agent 的能力扩展包，但官方市场没有精选和评测。我们从 23万+ Skills 中筛选出**真正好用**的那些，每周更新。

---

## ✨ 功能特性

- **25个精选 Skills** - 覆盖开发、写作、数据分析等场景
- **搜索筛选** - 按分类快速找到需要的 Skill
- **详细说明** - 每个 Skill 都有安装步骤、配置代码、适用场景
- **持续更新** - 每周新增优质 Skills
